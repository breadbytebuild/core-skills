# Technical Skills

The builder's toolkit. These skills cover the technical capabilities needed to go from idea to working product, including programming, data, AI, and the modern tooling landscape.

## Skills

- **Programming** — Write clean, maintainable, performant code
- **Data Analysis** — Extract insights from data using statistics and visualization
- **Prompt Engineering** — Design effective prompts for LLMs and AI systems
- **AI/LLM Literacy** — Understand AI capabilities, limitations, and best practices
- **No-Code/Low-Code** — Build functional products using visual and low-code tools
- **Growth Tooling** — Leverage analytics, A/B testing, and automation platforms
