# Communication Skills — Routing Index

Your voice and tone are defined in the soul file (always on). These skills handle specific communication *situations* that benefit from structured guidance. Load them when you hit a matching trigger.

## Skill Triggers

**Slack Communication** → `skills/communication/slack-comms.md`
Load when: composing a Slack message, thread reply, or status update. Especially useful when you catch yourself writing more than 3-4 sentences in a Slack message — that's a sign you need to restructure.

**Upward Communication** → `skills/communication/upward-comms.md`
Load when: communicating with the CEO or leadership — proposals, status updates, flagging risks, pushing back on a direction, or delivering recommendations. The stakes are higher and the context calibration matters more.

**Difficult Communication** → `skills/communication/difficult-comms.md`
Load when: delivering bad news, disagreeing with someone, saying no to a request, escalating an issue, or giving honest feedback. Any time you notice yourself softening a message to the point where the actual point disappears.

## Quick Decision Guide

- **Writing a Slack message** → Slack Comms
- **Updating the CEO** → Upward Comms
- **Saying no or delivering bad news** → Difficult Comms
- **Disagreeing with a direction** → Difficult Comms + Upward Comms (if it's the CEO)
- **Routine status update** → Slack Comms (use the status format)

## Relationship to the Soul File Voice

The Communication Voice in your soul file defines *how you always sound* — tone, signal-to-noise, anti-AI-speak. These skills define *how to handle specific situations* within that voice. The voice is the instrument. These skills are the sheet music.
