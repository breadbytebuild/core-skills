# Core Skills

A collection of skills to make AI agents world-class at specific capabilities.

## What This Is

Battle-tested thinking frameworks, build processes, and operational skills for AI agents. These skills were developed through real-world usage across software engineering, product building, mobile app development, video production, and agent operations.

Every skill has been through multiple iterations of use → failure → refinement. They encode hard-won lessons about what actually works.

## How to Use

1. **Start with `ROUTING.md`** — it's the master index that routes you to the right skill based on what you're doing.

2. **For substantive work**, load the relevant skill BEFORE starting. Skills define the process, anti-patterns, and quality bar.

3. **For simple tasks**, skip the routing. These skills are for work where quality matters.

4. **Combine skills** for complex tasks. Example: `pre-build-research` → `craft-loop` → `code-craft` → `deploy-verification` for a full build lifecycle.

## Key Skills

| Skill | Use When |
|-------|----------|
| **craft-loop** | Building anything significant. The iterative excellence loop. |
| **code-craft** | Writing, debugging, or reviewing code. |
| **deep-research** | Investigating unfamiliar domains. Resolving contradictions. |
| **pre-build-research** | Before any build. Understand the user and find best-in-class examples. |
| **deploy-verification** | After any deployment. Prove it works before saying "done." |
| **self-improvement** | Nightly reflection. Pattern extraction. Learning from mistakes. |
| **fleet-orchestration** | Delegating to sub-agents. Parallel workstreams. |
| **premium-app-design** | Building mobile UI that needs to be premium, not just functional. |
| **mobile-app-dev** | Full mobile app development — brainstorming to retention. |
| **remotion** | Programmatic video for social content, ads, shorts. |

## Philosophy

- **Skills are processes, not knowledge.** They encode HOW to do things well, not WHAT things are.
- **Anti-patterns are as important as patterns.** Every skill lists what NOT to do.
- **Generalized for any agent.** No company-specific references. Fork and customize for your domain.
- **Conservative evolution.** 3+ occurrences before promoting a pattern to a rule. Test with experiments. Revert what doesn't work.
